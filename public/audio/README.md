## Audio Format Recommendations
- Format: MP3 or OGG for web compatibility
- Sample Rate: 44.1kHz or 48kHz
- Bitrate: 128-192 kbps for good quality
- Each verse: 10-30 seconds duration
- Background: 2-5 minutes (will loop)
